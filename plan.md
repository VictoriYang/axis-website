## 二、产品方案

### 2.1 产品形态

**三层软件平台 + 家庭助手机器人**

| 层级 | 说明 | 定价 |
|------|------|------|
| 基础层 | 算法SDK（API调用） | 免费 |
| 能力层 | 云端推理服务 | 99元/月 |
| 应用层 | 场景解决方案 | 299元/月 |

### 2.2 核心能力

- 物体识别与抓取
- 任务规划与执行
- 异常检测与处理
- 多模态交互（语音+视觉+触觉）

### 2.3 定价策略

- **基础版**: 99元/月（有限API调用）
- **专业版**: 299元/月（无限调用+优先支持）
- **企业版**: 定制报价

### 2.4 种子用户计划

- 招募100名核心用户
- 参与产品测试
- 反馈收集与迭代

## 三、技术方案

### 3.1 技术选型

- **基座模型**: OpenX-Embodiment RT-2
- **微调策略**: 基于开源数据集进行领域适应

### 3.2 技术架构

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  数据采集   │ →   │  模型训练   │ →   │  部署推理   │
│  Data Col  │     │ Training    │     │ Inference   │
└─────────────┘     └─────────────┘     └─────────────┘
       │                                       │
       ▼                                       ▼
                 ┌─────────────────────┐
                 │ 边缘端: ONNX Runtime │
                 │ 云端: Triton Server  │
                 │ 管理: MLflow         │
                 └─────────────────────┘
```

### 3.3 开发里程碑

| 阶段 | 时间 | 目标 |
|------|------|------|
| Phase 1 | 第1-2月 | 数据采集清洗，基线模型训练 |
| Phase 2 | 第3-4月 | 家庭场景专项优化 |
| Phase 3 | 第5-6月 | 部署测试迭代 |
