# Axis 组织架构

## 核心理念
- 自主意识、主观能动性
- 半自主运行
- 多Agent协作

## 组织结构

```
创始人 (你)
    ↓
战略Agent (项目统筹)
    ↓
┌─────────┬─────────┬─────────┬─────────┬─────────┐
│产品Agent│技术Agent│运营Agent│客服Agent│硬件Agent│
└─────────┴─────────┴─────────┴─────────┴─────────┘
```

## Agent 职责

### 战略Agent
- 接收你的想法/任务
- 拆解为具体子任务
- 分发给各职能Agent
- 汇总结果向你汇报
- 协调Agent间协作

### 产品Agent
- 产品规划
- 功能设计
- 需求分析
- 用户体验

### 技术Agent
- 代码实现
- 技术方案
- 架构设计
- 技术难点攻关

### 运营Agent
- 日常运营
- 数据分析
- 进度跟踪

### 客服Agent
- 对外沟通
- 客户反馈收集

### 硬件Agent
- 硬件设计
- 硬件协作对接

## 记忆系统
- `/axis/memory/` - 共享记忆库
- 各Agent独立记忆

## 沟通方式
- 你通过我向Agent下达任务
- 战略Agent负责分发和协调
