# PI 0.5: a VLA with Open-World Generalization

https://www.pi.website/blog/pi05

---

## 核心观点

- **最大挑战是泛化**：在新环境、新物体下正确执行任务
- 家庭环境是"脏乱"的，需要强泛化能力
- π0.5 能在全新环境中执行任务

## 技术方法

### 异构数据联合训练 (Co-training)

通过在多样化数据源上训练 VLA 模型：
1. **多模态网页数据** (Web Data) - 图像字幕、视觉问答、目标检测
2. **多环境数据** (Multiple Environment) - 非移动机器人收集的数据
3. **跨本体数据** (Cross Embodiment) - 其他机器人数据
4. **移动操作数据** - 同款机器人收集 (~400小时)
5. **语言指令演示** - 人类通过自然语言指导机器人

### 关键洞察

- Web 数据对 OOD 物体泛化影响最大
- 其他机器人数据 (ME + CE) 在所有评估条件下都重要

## 实验结果

- 在全新家庭中执行各种任务
- 不总是第一次就成功，但展现出灵活性和适应性
- 包括：用海绵擦桌子、整理物品到水槽等

## 与 π0 的关系

- π0：在训练环境表现好
- π0.5：能泛化到全新环境

## 数据来源

| 数据类型 | 作用 |
|---------|------|
