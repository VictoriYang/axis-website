## 页面结构

### 1. 首页 Hero
- 全屏视觉，震撼开场
- Logo: AXIS
- Slogan: "Embodied Intelligence with Autonomous Will" / "具身智能 · 自主意志"
- 背景: 抽象神经网络/机器人轮廓动画
- CTA: "探索技术" / "Explore Our Technology"

### 2. 关于我们 About
**内容:**
- 一句话定位: "We build embodied intelligence with genuine autonomous will."
- 我们是谁: 一家专注具身智能的科技公司
- 愿景: 构建具有自主意识的智能体
- 核心理念: 自主意志、主观能动性

**布局:** 左侧文字，右侧抽象视觉

### 3. 技术架构 Technology
三个核心板块横向排列:

#### 3.1 具身大脑 (Embodied Brain)
- VLA (Vision-Language-Action) 模型
- 感知-理解-推理-决策
- 长期记忆与学习能力

#### 3.2 具身小脑 (Embodied Cerebellum)
- 运动控制与协调
- 实时反应与平衡
- 精细操作

#### 3.3 硬件层 (Hardware)
- 感知传感器
- 执行器
- 边缘计算

**布局:** 三列卡片，每个板块一个视觉icon

### 4. 核心技术 Core
- **VLA**: 端到端视觉语言动作模型
- **Sim2Real**: 仿真到真实环境迁移
- **Multi-Agent**: 多智能体协作
- **Continuous Learning**: 持续学习能力

**布局:** 网格卡片，每项技术一个卡片

### 5. 愿景 Vision
- 我们走向何方
- AGI终局
- "We believe true intelligence requires a body."

**布局:** 大字标语风格

### 6. 联系 Contact
- 合作招募
- 联系方式
- 社交媒体链接

**布局:** 简洁表单 + 邮箱

## 视觉规范

### 色彩
- 主色: #000000 (纯黑) / #FFFFFF (纯白)
- 强调色: #00D2FF (科技蓝)
- 辅助: #333333, #666666

### 字体
- 标题: 无衬线，厚重 (如 Inter Bold)
- 正文: 简洁清晰 (如 Inter Regular)
- 代码元素: 等宽字体

### 动效
- 滚动时渐入
- 鼠标悬停微动
- 背景微粒子/网格动画

---

## 响应式
- 桌面端 (>1024px): 完整布局
- 平板端 (768-1024px): 两列
- 移动端 (<768px): 单列堆叠

---

## 验收标准
1. 中英文双语切换流畅
2. 首屏加载 < 3秒
3. 动效流畅不卡顿
4. 移动端正常显示
