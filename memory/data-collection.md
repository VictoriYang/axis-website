---

## 数据流说明

| 阶段 | 说明 |
|------|------|
| 1. Oli SDK | 机器人原生 WebSocket 接口 |
| 2. ROS2 Bridge | 桥接层，话题转换 |
| 3. 话题订阅 | 视觉 + 状态 + 指令 |
| 4. 采集录制 | MCAP 落盘 |
| 5. 后续处理 | 标注工具 / 数据清洗 |
